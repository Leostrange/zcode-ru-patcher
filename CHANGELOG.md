# Changelog — ZCode-Ru-Patcher v6.3

Все изменения в рамках одной версии **v6.3**.

---

## 2026-06-25

### Changed
- **UI-локализация** — переведены интерфейсные строки патчера
  (кнопки, заголовки, сообщения, диалоги).

## 2026-06-26

### Changed
- **Локализация скиллов/инструментов** — переведены описания и названия
  скиллов, доступных в патчере.
- Исправлены **непереведённые строки** — часть UI-элементов оставалась
  на английском после предыдущих раундов локализации.
- Патчер пересобран на **NSIS без MUI** (silent self-extractor).
  Бинарный размер вырос с ~76 MB до ~107 MB — теперь все файлы патча
  включены как единый NSIS-пакет, а не внешний архив.
- Убран диалоговый интерфейс MUI — патчер работает в полностью
  автоматическом (silent) режиме.
- Исправлена ошибка **"Error opening file for writing"** при распаковке
  патча.
